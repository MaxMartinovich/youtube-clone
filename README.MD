# Youtube-Clone(Página Principal)

# Tentativa de Clonar o Youtube(Página Principal) 

## Descrição
Primeiro site que eu criei, tentei recriar de maneira exata algumas funcionalidades do Youtube, como os botões, a sidebar, a barra de pesquisa, além de colocar as thumbnails e os videos



## Tecnologias utilizadas
- HTML
- CSS
- JavaScript

## Funcionalidades
- Ao clicar no video ou no titulo, leva diretamente ao video
- Ao clicar no icone do canal ou no nome do canal, leva para a pagina do canal
- Ao passar o mouse por cima dos icones, mostra o tooltip de cada um deles
- A Header não sai da página mesmo que o usuário scrolle para baixo, ela continua no topo sem passar por cima dos videos 

## Como executar o projeto
1ª - Instale alguma IDE de sua escolha, recomendo VSCODE para trabalhar com JavaScript,HTML e CSS

2ª - Copie o código disponibilizado

3ª - Execute a aplicação!
